# Personal-Homepage
黄嘉祥的个人主页，使用Flask框架搭建。

1. **安装环境**

   确保服务器上已经安装python环境或者anaconda。

   安装flask和gevent

   ```pip
   pip install flask
   pip install gevent
   ```

2. **运行服务端**

   在目录下控制台中运行app.py

   ```terminal
   python app.py
   ```

   在浏览器中访问本地回环地址127.0.0.1或者在其他设备上访问服务端的公网ip
